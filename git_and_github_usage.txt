STEP 1>Download git by searching git download.

(In any terminal)
STEP 2>Only for first time create git username and gmail with following steps.

        STEP a>git config --global user.name "abcxyx"(for example).
        STEP b>git config --global user.email "abcxyz@pqr.com"(for example).

STEP 3>Check with following steps.

        STEP c>git config --global user.name.
        STEP d>git config --global user.email.

STEP 4>Use mkdir and cd with foldername to rich folder of mainfile(for example = multiply.java).

STEP 5>Write git init to initialize the folder.
